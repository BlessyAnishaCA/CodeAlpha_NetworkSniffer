\# CodeAlpha Network Sniffer



A basic network packet sniffer built with Python and Scapy.



\## Requirements

\- Python 3.x

\- Scapy: pip install scapy

\- Npcap (Windows): https://npcap.com



\## How to Run

Run as Administrator in terminal:

python sniffer.py



\## Features

\- Captures live network packets

\- Shows source/destination IP and ports

\- Identifies TCP, UDP, ICMP protocols

\- Shows payload preview

\- Saves capture to .pcap file
